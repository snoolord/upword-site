Technologies

Upword is rebuilt with Vanilla JavaScript from React. The bundle size decreased from 1 megabyte to 100 kb from this transition. Upword Server was created to store words and keep track of popular words.

Upword

JavaScript, HTML, CSS, Webpack

Upword Server

Express.js, MongoDB, Async.js, Mongoose

Webster API

Upword Server makes API calls through look-upword to save words to the database thus minimizing API calls

look-upword

I converted word data from XML to JSON and made an NPM package for others to use
